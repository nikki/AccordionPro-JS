Hey Carlos!

To get the plugin working with Ajax, you'll need to use the unminified version of Accordion Pro JS (the file is called jquery.accordionpro.js) and comment out lines 668 and 686.

If you view source on this page, you can see the Ajax call. ajax-template.html is the ajax template.

I wouldn't call this a bug as such, more an unforeseen use case - you're the first person that's ever wanted to create an accordion via Ajax :)

Hope this helps!
